
## Modelos de negocio

| Nombre                             | Ejemplos                              | Como funciona                                                                                                                                                                                                   | Cobro                                                                                                            |
| ---------------------------------- | ------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------- |
| El SaaS Puro o "Self-Service"      | Netflix, Canva, SunFast               | El cliente entra a tu web, pone su tarjeta de crédito y empieza a usarlo (automático)                                                                                                                           | Suscripción mensual<br>($15/mes aprox)                                                                           |
| El SaaS Empresarial o "High-Touch" | Sistemas de ventas, restaurantes, etc | Tienes que configurar la aplicación para la empresa (base de datos, usuarios, configuraciones, etc)                                                                                                             | Costo implementación (pago único: $600 aprox)<br><br>Suscripción mensual<br>($40 aprox)                          |
| Software Exclusivo "Caja Blanca"   | Sistemas a medidas                    | Le creas una aplicación a su medida, pero los derechos de autor le pertenece a Software Lion                                                                                                                    | Costo por desarrollo<br>($2600)<br><br>Soporte/mantenimiento mensual<br>($50 aprox)                              |
| Propiedad Intelectual Completa     | Sistema y código fuente a medida      | Vender el software, y el codigo fuente, si luego quieres desarrollar un Saas tienes prohibido "Copiar y Pegar" debes desarrollarlo todo desde 0, la idea puede ser la misma, tienes prohibido copiar el código. | Costo por desarrollo + propiedad intelectual<br>($8500)<br><br>Soporte/mantenimiento<br>opcional ($50 aprox)<br> |
